ThingSpeak Configuration for LoRa Tunnel Safety System:

Platform: https://thingspeak.com

Steps:
1. Create an account and a new channel.
2. Add fields like:
   - Temperature
   - Gas Levels
   - Worker ID
   - Tunnel Section
3. Get your Write API Key from the channel settings.
4. In your Raspberry Pi (or any device), use HTTP POST to send data using this format:
   https://api.thingspeak.com/update?api_key=YOUR_KEY&field1=value1&field2=value2

Data is sent from the receiver via Python or serial interface to ThingSpeak.
